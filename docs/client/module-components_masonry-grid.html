activate()

Activate the masonry grid and sort all children into columns

append(…nodes)

Append nodes to the masonry grid. If the masonry grid is active, it will sort them into the columns in-order.

Parameters:
Name Type Attributes Description
nodes Node <repeatable>

Nodes to be appended

deactivate()

Deactivate the masonry grid, moving all elements out of columns in-order and removing the grid columns
